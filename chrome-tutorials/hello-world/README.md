# Hello World Extension

A basic Chrome extension demonstrating the fundamentals of extension development.

## Description

This is a minimal Chrome extension that displays a simple popup when the extension icon is clicked. It serves as a starting point for learning Chrome extension development and understanding the basic structure of a Manifest V3 extension.

## Features

- Simple popup interface
- Logs a greeting message to the console
- Demonstrates basic extension structure with manifest.json

## Files

- `manifest.json` - Extension configuration and metadata
- `hello.html` - Popup HTML interface
- `popup.js` - Popup script that logs to console
- `package.json` - npm configuration for Chrome types

## Usage

1. Click the extension icon in your Chrome toolbar
2. A popup will appear with the Hello World interface
3. Open the browser console to see the greeting message

## Development

This extension includes Chrome types for TypeScript/JavaScript development:

```bash
npm install
```
